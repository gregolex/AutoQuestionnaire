# AutoQuestionnaire
Displays Statistical analysis for Users, Questions, ID's, and Answers.

In order to properly compile this program follow these steps:
1. Download folder AutoQuestionnaire/src/assignment1/cs3560/gregkalaydjian/
2. Create new Java project on IDE (i.e. Netbeans)
3. Copy files into project: Question.java, SimulationDriver.java, Student.java, VotingService.java
4. Rename package on the top of each java file
5. Run SimulationDriver.java
